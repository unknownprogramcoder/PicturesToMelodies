## 요구사항
소스 코드를 실행하려면 아래의 소프트웨어가 설치되어 있어야 합니다.

* 파이썬 3.x
* NumPy
* Matplotlib
 
또한 선택사항으로 다음 라이브러리를 사용합니다.

* SciPy
* CuPy


## 실행방법

각 장의 디렉터리로 이동한 후 python 명령을 실행하세요(**다른 디렉터리에서는 제대로 실행되지 않을 수 있습니다!**).

```
$ cd ch01
$ python train.py

$ cd ../ch05
$ python train_custom_loop.py
```
